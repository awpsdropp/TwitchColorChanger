## TwitchColorChanger &mdash;

### Change color of your name on twitch using delay.

#

### installation:

- download [nodejs](https://nodejs.org/en) lts version.
- download repo.
- change settings in index.js file.

#

### install packages:

- `npm i -g yarn`
- `yarn`

#

### run code:

- `yarn dep --- for deploy`
- or
- `yarn dev --- for development`
